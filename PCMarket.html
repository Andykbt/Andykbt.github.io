Description

PCMarket is an e-commerce store built using HTML, CSS, and JQuery.
I took over this project after a previous developer had already built the website.
When I took over, major plans were in place to redesign and change many different
functionalities of the website. I was tasked to implement these changes, learn
the Neto B@se templating language and get used to the current codebase quickly.

Neto's themeing and templating mainly uses HTML, CSS, and JQuery.
Products, categories, and other content could be fetched from the backend by using
Neto's B@se templating language which is used in HTML files to display content.

Some internal projects also required me to write scripts to automate repetitive tasks.
Such scripts included:

  • Script to fetch product data from Neto and create CSV files easily
  • Python script that automatically updated the Algolia indices
  • An automatic review importer to Stamped.io
